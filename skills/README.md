# Chronicle Skills

Reusable pipeline stages for The Human Chronicle. Each skill is a
self-contained markdown file with clear inputs, steps, and outputs.

## Pre-Pipeline

| Skill | Purpose |
|-------|---------|
| [chronicle-landscape](chronicle-landscape.md) | Write a region LANDSCAPE file — ASCII map, peoples table, connections |
| [chronicle-overview](chronicle-overview.md) | Write a people OVERVIEW file — their story from inside their worldview |

## Chapter Pipeline

| Skill | Stage | Purpose |
|-------|-------|---------|
| [chronicle-opening](chronicle-opening.md) | 1 OPENING | Write a hook scene (~800 words) — one person, one place, one moment |
| [chronicle-gate](chronicle-gate.md) | 2/8 GATE | Score against The Gate (30pts) or The Chronicle (60pts) rubric |
| [chronicle-panel](chronicle-panel.md) | 3/5 PANEL | Run 5-voice panel review — Tuchman, Ibn Khaldun, Achebe, Davis, Kapuscinski |
| [chronicle-notes](chronicle-notes.md) | 4 NOTES | Write the production bible — sources, arc, section plan, panel risks |
| [chronicle-chapter](chronicle-chapter.md) | 7 WRITE | Write a full chapter (~5000 words) + figures — all 13 style principles |
| [chronicle-clean](chronicle-clean.md) | 9 CLEAN | Run 3-lens editorial pass — Judgment Auditor, Voice Keeper, Compression Editor |
| [chronicle-board](chronicle-board.md) | 10 BOARD | Create domain experts and run accuracy/authenticity review |

## Orchestration

| Skill | Purpose |
|-------|---------|
| [chronicle-e2e](chronicle-e2e.md) | Full 11-stage pipeline end-to-end — OPENING through FINAL |

## Quality & Evolution

| Skill | Purpose |
|-------|---------|
| [chronicle-inside-check](chronicle-inside-check.md) | Audit for outside-perspective violations — the Achebe Test, sentence by sentence |
| [chronicle-innovation](chronicle-innovation.md) | Scout locked chapters for rubric innovations — the rubric evolves upward |
| [chronicle-cross-compare](chronicle-cross-compare.md) | Compare chapters across cultures — find patterns that improve the system |
