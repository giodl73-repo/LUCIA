# PR-B2 — Structural Engines as Cross-Cultural Narrative Types: A Taxonomy From a 115-Chapter Corpus

**Track**: B (The Artifact's Own Dimensions)
**Position in chain**: Second paper — requires B1's validation method; gates B3.
**Status**: planned, round 0

---

## Hypothesis

Chapter-arc structure is empirically typologizable into a bounded set of Structural Engine types (Covenant, Material Object, Contradiction-Held-as-Engine, Non-Chronological, Precondition, Administrative Machine, etc.), each instantiated across culturally-different chapters. The taxonomy is cross-cultural rather than region-specific.

## Quantification Contract

| Component | Value |
|-----------|-------|
| Primary number | Engine-type count with culture-breadth metric: mean cultures-per-engine-type, median regions-per-engine-type, non-provincialism score (engines with ≥ 3 instances across ≥ 3 regions). |
| Experiment design | Walk INNOVATIONS.md and all locked chapters. For each engine type, enumerate instances and compute breadth. |
| Decision it changes | Whether structural engines are cross-cultural patterns or region-specific artifacts. |
| Null fallback | Mean cultures-per-engine ≤ 2 → engines are not a taxonomy; reframe as case studies. |
| Falsification condition | Non-provincialism score (≥ 3 regions) < 50% of engine types → engines are region-bound, not universal. |

## Outline

1. **Introduction.** Structural engine vs. plot. The rubric's v1.1 amendment. Why engines are a scale-above-voice typology.
2. **The engine types.** Formal taxonomy:
   - Covenant (v1.1)
   - Material Object as Structural Principle (v1.5)
   - Grief / psychological mechanism
   - Non-Chronological Organizing Principle (v1.5)
   - Precondition Engine (single-instance: Luba)
   - Portable Civilization (single-instance: Romani)
   - Administrative Machine (Canada: Indian Act)
   - Contradiction-Held-as-Engine (v1.7, cluster of 5 colonial empires)
3. **Validation.** For each type, enumerate instances, compute culture breadth, compute regional breadth. Non-provincialism score.
4. **Cross-type analysis.** Which engines are most broadly cross-cultural? Which are culture-type-specific (oral traditions, colonial, theocratic, etc.)?
5. **Three case studies.** One pre-modern (Covenant / Mali), one mid-modern (Material Object / Norse), one modern (Contradiction-Held / French Empire).
6. **Discussion.** Engine as narrative-arc primitive. Comparison to Propp, Campbell, other narrative typologies.
7. **Falsification.** Non-provincialism threshold.

## Anticipated Failure Modes

- Some chapters have dual engines (e.g., Brazil: political substrate-exhaustion + knowledge vindicating). Classify with primary + secondary; report both.
- Single-instance engine types (Precondition, Portable Civilization) drag breadth metrics down; stratify and report.
- "Engine" is formally defined in the rubric — do not mix with generic "plot" discussions in the prose.
