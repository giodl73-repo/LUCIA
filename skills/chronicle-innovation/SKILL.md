You are running /chronicle-innovation for: {{chapter}}

Scout a locked chapter for rubric innovations — exceptional techniques the
rubric did not anticipate. The rubric evolves upward through what the best
chapters teach it.

---

## Input

- **Chapter**: Path to a locked (FINAL) `chapter.md`, or multiple chapters

---

## What Counts as an Innovation

An innovation is a technique that:
1. Raises the bar beyond what the rubric's current anchors describe
2. Cannot be captured by simply giving a higher score on an existing dimension
3. Represents a genuinely new way of achieving the rubric's goals

Innovations are NOT:
- High scores on existing dimensions (that's just good writing)
- Techniques already described in the Style Guide
- Personal aesthetic preferences

---

## Step 1 — Read for Surprise

Read the chapter looking for moments where the writing does something
the rubric framework did not predict. Ask:

- Did this chapter solve a problem the rubric does not address?
- Did it achieve immersion, evidence, progression, figures, voice, or
  landing through a mechanism the scoring anchors do not describe?
- Did it find a way to handle a tension between panel voices that the
  system had not anticipated?

---

## Step 2 — Name the Technique

For each innovation, describe:
- **What the chapter did** — the specific technique, with a quoted example
- **Which dimension it touches** — Scene, World, Honesty, Texture, Voice,
  Pull, Immersion, Evidence, Progression, Figures, Landing
- **Why the current rubric misses it** — what would need to change to
  capture this technique
- **Whether it is culture-specific or universal** — does it only work for
  this type of culture, or is it a general improvement?

---

## Step 3 — Check Innovation Threshold

Read `scoring/INNOVATIONS.md`. If 2+ innovations have now appeared in the
same dimension or category, flag a rubric amendment proposal:

```
AMENDMENT PROPOSAL: [dimension]
Innovations: [list the 2+ innovations]
Proposed change: [what should the rubric add or revise]
Impact: [how this changes scoring for future chapters]
```

---

Append entries to: `scoring/INNOVATIONS.md`

Entry format:
```markdown
## [Date] — [People] — [Era]

**Dimension:** [Which rubric dimension]
**Example:** "[Quoted passage from the chapter]"
```

If amendment threshold reached, include the amendment proposal at the end.
