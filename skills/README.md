# Chronicle Skills

Reusable pipeline stages for The Human Chronicle. Each skill lives in
its own directory as `SKILL.md`.

## Pre-Pipeline

| Skill | Purpose |
|-------|---------|
| [chronicle-landscape](chronicle-landscape/SKILL.md) | Write a region LANDSCAPE file — ASCII map, peoples table, connections |
| [chronicle-overview](chronicle-overview/SKILL.md) | Write a people OVERVIEW file — their story from inside their worldview |

## Chapter Pipeline

| Skill | Stage | Purpose |
|-------|-------|---------|
| [chronicle-opening](chronicle-opening/SKILL.md) | 1 OPENING | Write a hook scene (~800 words) — one person, one place, one moment |
| [chronicle-gate](chronicle-gate/SKILL.md) | 2/8 GATE | Score against The Gate (30pts) or The Chronicle (60pts) rubric |
| [chronicle-panel](chronicle-panel/SKILL.md) | 3/5 PANEL | Run 5-voice panel review — Tuchman, Ibn Khaldun, Achebe, Davis, Kapuscinski |
| [chronicle-notes](chronicle-notes/SKILL.md) | 4 NOTES | Write the production bible — sources, arc, section plan, panel risks |
| [chronicle-chapter](chronicle-chapter/SKILL.md) | 7 WRITE | Write a full chapter (~5000 words) + figures — all 13 style principles |
| [chronicle-clean](chronicle-clean/SKILL.md) | 9 CLEAN | Run 3-lens editorial pass — Judgment Auditor, Voice Keeper, Compression Editor |
| [chronicle-board](chronicle-board/SKILL.md) | 10 BOARD | Create domain experts and run accuracy/authenticity review |

## Orchestration

| Skill | Purpose |
|-------|---------|
| [chronicle-e2e](chronicle-e2e/SKILL.md) | Full 11-stage pipeline end-to-end — OPENING through FINAL |

## Session Management

| Skill | Purpose |
|-------|---------|
| [chronicle-handoff](chronicle-handoff/SKILL.md) | End-of-session handoff — capture state, update punchlist, prepare for next session |

## Quality & Evolution

| Skill | Purpose |
|-------|---------|
| [chronicle-inside-check](chronicle-inside-check/SKILL.md) | Audit for outside-perspective violations — the Achebe Test, sentence by sentence |
| [chronicle-innovation](chronicle-innovation/SKILL.md) | Scout locked chapters for rubric innovations — the rubric evolves upward |
| [chronicle-cross-compare](chronicle-cross-compare/SKILL.md) | Compare chapters across cultures — find patterns that improve the system |
