## B-3 — Raymond Westbrook — Board Review of Hammurabi's Babylon (Era 1)

**Expertise relevant to this chapter:** Comparative ancient Near Eastern law; editor of A History of Ancient Near Eastern Law; specialist in the debate over whether the code was a functioning law code or a royal monument.

**Historical accuracy:**
- The specific cases cited from the code are accurate: the false accusation case (Law 1), the theft case (Law 6), the physician's liability (Law 218), the builder's liability (Laws 229-230), the goring ox (Laws 250-252), the tavern-keeper (Law 108), the merchant-agent relationship (Laws 100-107), the naditu inheritance (Laws 178-184), the eye-for-eye (Laws 196, 198, 199). All correctly represented.
- The class structure (awilum/mushkenum/wardum) is accurately described. The proportional penalty structure is correct.
- The prior law codes (Ur-Namma, Lipit-Ishtar) are correctly noted as predecessors.

**Cultural authenticity:**
- The chapter presents the code as public theology, which is one valid interpretation. However, the chapter should acknowledge more explicitly that the scholarly debate about the code's function is unresolved. Was it a reference text for judges? A royal monument? A scribal exercise? A combination? The chapter leans toward the "royal monument / public theology" interpretation, which is legitimate, but the debate deserves a sentence or two.
- The naditu women's legal practice is accurately described and correctly linked to the code's provisions. This is a strong example of the code in action.

**What the chapter gets right:**
- The emphasis on the code's publicness as its defining innovation (not the content of the laws, which drew on existing tradition, but the act of public publication) is a genuinely historical insight.
- The epilogue's curse is correctly presented as the mechanism that binds the future to the stele.
- The class distinctions are presented without modern judgment, which is correct.

**What needs correction:**
- The chapter states that "what Hammurabi did that none of them had done was to make the publication itself the central act of kingship." This is slightly overstated. Ur-Namma's code also had a theological prologue linking law to divine mandate. Hammurabi's innovation was scale and prominence, not the concept itself. A slight qualification would be more accurate.
- The relationship between the code and actual legal practice deserves acknowledgment. The chapter's notes.md discusses this debate, but the chapter itself does not. A brief passage noting that actual court records from the period do not always follow the code's prescriptions would strengthen the chapter's honesty.

**Verdict:** APPROVED-WITH-CONDITIONS (two minor qualifications recommended)
