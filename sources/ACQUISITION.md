# LUCIA Source Acquisition Punchlist

**Purpose:** Tracks sources that have been identified as needed but not yet acquired. Items are added here when a chapter's `notes.md` file contains a `NEEDED:` entry — a source the author knows they require but does not yet have in hand.

---

## How Items Get Added

When writing or researching a chapter, if you identify a source you need but don't have:

1. Add a `NEEDED:` entry in the chapter's `notes.md` file:
   ```
   NEEDED: [Author/Title/Description] — [why it's needed] — [priority P1/P2/P3]
   ```
2. Add a corresponding row to the punchlist table below.
3. When the source is acquired and verified, remove it from this list and add it to `MASTER.md` and the relevant `SOURCES.md`.

---

## Priority Levels

| Priority | Meaning |
|----------|---------|
| P1 | Blocks a chapter currently in progress. Acquire immediately. |
| P2 | Needed for an upcoming chapter. Acquire before drafting begins. |
| P3 | Nice to have — would strengthen coverage but chapter can proceed without it. |
